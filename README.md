# Ultra-Lightweight SaaS Starter (Next.js + Preact)

A production-ready, ultra-lightweight starter kit for building SaaS dashboards and web apps.  
Powered by **Next.js App Router + Preact + UnoCSS + Zustand + Valibot + uPlot**.

## Tech Stack

- Next.js 14 (App Router)
- Preact (React drop-in replacement)
- UnoCSS (atomic CSS)
- Zustand (global state)
- Valibot (validation)
- uPlot (charts)
- Ky (HTTP client)
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
